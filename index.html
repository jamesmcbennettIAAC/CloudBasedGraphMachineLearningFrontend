THESIS

Cloud-Based Graph Machine Learning (CBGML)

IAAC MaCAD 2022/23 Thesis

by Ren Rainville & James McBennett

Thesis Supervisor: David Andrés León

This thesis applies knowledge gained from two IAAC MaCAD 2022/23 courses.

CLOUD-BASED DATA MANAGEMENT emphasized front-end development using HTML, CSS, and JavaScript to interface with Rhino.Compute to control Rhino3D or Grasshopper3D geometry within a web browser.

GRAPH-MACHINE LEARNING utilized graphs composed of edges and nodes to represent buildings. Machine learning techniques were employed to train a model for predicting node and edge classification.

This thesis centers on PREDICTING EGRESS FOR MULTI-STORY RESIDENTIAL BUILDING through graph machine learning (GML). The process commences with users outlining building geometry on a leaflet.js map. Coordinates are then processed, including scaling, repositioning to the origin, and rearranging counter-clockwise, followed by selecting floor count and corridor type. This data is then sent as a JSON to the back-end.

Special thanks to David Andrés León, Justyna Szychowska, Dai Kandil, Sara Kessba, and Lucas Sentís Fuster.
